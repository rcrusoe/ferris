Installution:

  1. Installing the theme
  2. Unzip the downloaded file from Themeforest you have purchased
  3. The zip containing html documentation folder and theme folder 'forest theme'
  4. Upload the theme folder to '/ghost/content/themes/'
  5. Restart Ghost
  6. Activate the theme trough Ghost backend

Content Formatting

(only availabe in Forest theme)

Intro paragraph:

<p class="intro">
text...
</p>

Pulling content:

.pull-left
.pull-right
.pull-center

Example:

More example with code:

Pull center image with text

    <img src="/content/images/2015/04/brook-forest-hills-3939.jpg" alt="">
    <figcaption>Forests are the dominant terrestrial ecosystem on Earth</figcaption>
</figure>
