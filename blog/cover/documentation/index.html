1 . Getting Started

For the purposes of this documentation, it is assumed that you have Ghost up and running. If not, please visit support.ghost.org and follow the guide to install and configure Ghost.

Please make sure you are running the latest version of Ghost before installing & using the theme.

2. Installation

  1. Navigate to the directory on your server where Ghost is installed.
  2. Once there, navigate to the /content/themes/ directory.
  3. Upload the theme files to this directory.
  4. Restart Ghost.
  5. Log into your Ghost Admin.
  6. Navigate to Settings > General and select the theme in the theme options dropdown menu.
  7. Click 'Save'.

Note: If you're on using Ghost(Pro), please follow these instructions on how to install a theme.

3. Upgrading

If you have made any changes to the template or SCSS/CSS files, you will need to re-integrate your custom theme changes.

Always perform a backup of the theme files before upgrading

  1. Download the latest version of the theme from your purchases page on Creative Market.
  2. Navigate to the folder on your server where Ghost is installed.
  3. Once there, navigate to the /content/themes/ folder.
  4. Upload and replace all of the older theme files with the newer ones.

Once you update your theme, you will want to restore your changes to the theme-options.js file.

4. Settings

In Settings > General several options will alter the content in the theme:

  • Blog Logo - Appears as the first item the header. We recommend a maximum size of 240 x 30 pixels.
  • Blog Title - The blog title will be displayed if no blog logo has been uploaded.
  • Blog Description - Appears as your blog description in the search engines.
  • Blog Cover - This theme does not use or display the cover photo.
  • Posts per page - This is how many posts are displayed per page. We recommend setting this to 12 for the theme to display an equal number of items per page.

5. Customization

If you're comfortable with HTML & CSS, you can make any changes to the theme by editing the template files.

Please be aware that when the theme is updated in the future, you will need to re-integrate your custom theme changes.

The CSS files were created using SASS. If you are familiar with SASS, you can easily customize the theme.

Using CSS

You can add your own custom CSS by using the Code Injection feature in Ghost. All you need to do is put all your custom CSS in the Blog Header section.

This is the preferred method of customization, because it doesn't involve editing any theme files.

Note: Don't forget to wrap your all your CSS in <style> tags.

Using SASS

You will find the following SCSS files in /assets/scss/ directory:

  • screen.scss - This is the main CSS stylesheet. It imports the mixins.scss and variables.scss files.
  • mixins.scss - This contains all the CSS declarations are reused throughout the theme.
  • variables.scss - This contains all the variables that you can tweak to customize the way the theme looks.
  • normalize.scss - This contains Normalize.css, which is a customisable CSS file that makes browsers render all elements more consistently and in line with modern standards.

Note: You will need a SASS preprocessor, such as Grunt to use the SCSS files.

Navigation

You can customize the navigation menu by using the Navigation builder in your Ghost admin area.

  1. Click Settings in your Ghost admin area.
  2. Click Navigation from the menu on the left.
  3. Add, remove and/or reorder navigation items on the right.
  4. Click save.

Social Media

Add your own social media links to the header by editing the theme-options.hbs file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.
  3. Find the Social Media section

  4. Add your social media links (between the quotation marks) to the corresponding social media variables.

    Facebook Example:

     var facebook_link = "http://www.facebook.com/my_facebook_username";

  5. Upload the changed theme-options.js file to the partials > js folder.

Note: If you don't want a specific social media icon to appear, just leave variable blank.

var facebook_link = "";

Note: If you wish to show RSS icon, change the rss_link value to true.

var rss_link = true;

Favicon

You can add you own custom Favicon to your blog by replacing the favicon.ico file located in the assets directory:

  1. Navigate to the assets directory on your server.
  2. Replace the favicon.ico file with your own custom icon.

Post Images

You can add a post image to your posts by adding one in the post settings. This post image will be displayed along with the title of the post on the homepage and in the post itself.

  1. Login to the Ghost Admin area.
  2. Create or edit a post.
  3. Click the post settings icon, this is located next to the publish/update post button.
  4. Click "Add post image.
  5. Save/update the post.

Post Layouts

You can customize the layout of your posts with these included post layouts. These post layouts will affect the entire post giving it a unique look from the standard post layout.

One column layout
  1. Add a one-column tag to the post.
  2. Save/update the post.
Two column layout
  1. Add a two-column tag to the post.
  2. Save/update the post.

Note: If you want to revert back to the standard post layout just remove the tag that was added to the post.

Comments

You can enable Disqus comments on your blog by adding your Disqus shortname to the theme-options.js file located in the assets > js folder:

  1. Add your site to your Disqus account and setup a unique Disqus URL. The unique shortname you pick for your Disqus URL will be important to remember!
  2. Navigate to the assets > js folder on your server.

  3. Open the theme-options.js file and add your unique shortname to the disqus variable.

    Example:

     var disqus = "DISQUS_SHORTNAME_HERE";

  4. Save/update the theme-options.js file.

Note: If you don't want comments on your blog just leave the disqus variable blank.

var disqus = "";

Image Lightbox

This feature will automatically add an image lightbox to all images within a post.

You can enable it by editing the theme-options.js file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.

  3. Find the lightbox variable and change the value from false to true.

    Example:

     var lightbox = true;

  4. Upload the changed theme-options.js file to the partials > js folder.

Note: If you wish to disable this feature just change the true value to false.

var lightbox = true;

Infinite Scroll

You can enable infinite scroll by editing the theme-options.js file located in the partials > js folder.

  1. Navigate to the partials > js folder on your server.
  2. Open the theme-options.js file in your favorite text editor.

  3. Find the lightbox variable and insert the text (between the quotation marks) scroll or click. Here are the differences between the two options:

    Scroll will load new posts while scrolling:

     var infinite_scroll = "scroll";

    Click will load new posts with a click of a button:

     var infinite_scroll = "click";

  4. Upload the changed theme-options.js file to the partials > js folder.

Note: If you wish to disable infinite scroll just change the infinite_scroll variable blank.

var infinite_scroll = "";
